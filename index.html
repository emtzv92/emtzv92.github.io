This is a website created by me and other collegues, which lets users register, create an account and play Prisoner's Dilemma.

Prisoner's Dilemma is a situation in which two players each have two options whose outcome depends crucially on the simultaneous choice made by the other, often formulated in terms of two prisoners separately deciding whether to confess to a crime.

Magik 8 Ball

This app is intended to imitate the functions of a Magic 8 Ball.

This app was created using android studio, based on the Magic 8 Ball. This app contains the launch icon, and before the actual app starts, s splash screen shows up first about the author and how it works. The main activity page, asks the user yes or no questions. Flip the device down to ask it a question and flip it back up to get a response. This app utilizes the deivce's sensors to determine whether the device is pointing up or down.
